Covid-19

The Covid-19 pandemic was a devastating and unique event in our history. It created lasting transformations for society and still remains a controversial topic. Opinions range on masks, effectiveness of lock-down mandates, the lethality of the virus, and every facet of the pandemic.
This presentation will examine the data set of number of deaths of individuals infected with Covid-19 at the time of their death. This data set was produced by the U.S Department of Health & Human Services and is available at this link. The data for 2020 and 2021 is final while 2022 and 2023 are provisional.
The presentation will contain another data set to supplement the Covid-19 data set with national and state wide population figures. The presentation will use the 2020 census from the United States Census Bureau for population figures, available at this link.

Questions

  • Which condition contributed to more deaths
  • How age is related to deaths
  • Which year had the most deaths
  • Death-rate per month
  • Total deaths per State
  • Percentage of deaths per State

create_total_to_date <- function(data_V) {
  new_V <- vector("integer", length(data_V))
  new_V[1] = data_V[1]
  for(x in 2:length(data_V))
    new_V[x] = new_V[x - 1] + data_V[x]
  return(new_V)
}
